About ForestSathi

An AI-powered wildfire risk assessment platform built specifically for Nepal's unique geographical and ecological challenges.

What is ForestSathi?

ForestSathi (Forest Companion) is an intelligent wildfire risk prediction system designed to help communities, forest officials, and researchers understand and mitigate fire risks across Nepal's diverse terrain. The platform combines satellite data analysis with machine learning to provide real-time, location-specific fire risk assessments.

Using machine learning models trained on over 190,000 historical fire events from NASA's FIRMS satellite data, ForestSathi provides location-specific risk assessments with 98.7% accuracy. The system analyzes multiple factors including geographical coordinates, seasonal patterns, vegetation type, and historical fire frequency to generate accurate predictions.

The platform features an interactive heatmap visualization showing fire hotspots across Nepal, yearly fire statistics for any searched location, and detailed regional analysis covering Nepal's three ecological zones: Terai Plains, Mahabharat Range (Hills), and High Himalayas.

Technology Stack

ForestSathi is built using modern web technologies and machine learning frameworks to deliver fast, accurate, and reliable wildfire risk predictions. The entire stack is optimized for performance and deployed on cloud infrastructure.

Random Forest AI (Scikit-learn)
Machine learning model trained on 190K+ fire events with 98.7% prediction accuracy. Uses ensemble learning for robust risk classification.
Leaflet.js + Heatmap Layer
Interactive mapping library with real-time heatmap visualization. Displays 15,000+ fire hotspots with intensity-based coloring.
Geopy + Nominatim Geocoding
OpenStreetMap-based geocoding with 90+ Nepal locations database fallback for reliable location search.
Flask + Gunicorn (Python)
Production-ready Python backend with RESTful APIs. Deployed on Render cloud with auto-scaling.
Pandas + NumPy (Data Processing)
High-performance data analysis libraries for processing 190K+ satellite fire records efficiently.
HTML5, CSS3 & JavaScript
Modern responsive frontend with CSS animations, dark theme, and mobile-first design approach.
